Event Booking Management System

A web-based event booking management system built using HTML, CSS, JavaScript, and PHP. This project allows users to browse, book, and manage event reservations efficiently.

✨ Features:
User authentication (registration & login)

Event listing with details

Booking and cancellation system

Admin panel for event management

Payment gateway integration (optional)

Responsive design for all devices

🛠️ Tech Stack:
Frontend: HTML, CSS, JavaScript

Backend: PHP, MySQL

Database: MySQL
